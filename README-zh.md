<h1 align="center">KookyBotChatBotTemplate</h1>

<p align="center">
  <b>基于 Kotlin 和 KookyBot 的聊天机器人的开发模板</b>
</p>

本项目的开发旨在开发一个在使用 Kotlin 和 KookyBot 创造的聊天机器人时可以复用的代码库。

本项目已经加入了一些功能，例如：

- 通过文件进行配置
- 国际化（i18n）
- KookyBot 监听器的模板
- 通过 HTTP API 进行监控状态和控制聊天机器人的可能性
